# IMPACTO DE LAS REDES SOCIALES EN LA SALUD MENTAL DE LOS ADOLESCENTES
##  DESCRIPCION
Este proyecto consiste en un **Análisis Exploratorio de Datos (EDA)** centrado en estudiar cómo el uso de las redes sociales puede influir en distintos aspectos de la salud mental de los adolescentes.

A través de diferentes visualizaciones, análisis estadísticos e hipótesis se exploran relaciones entre:

- Tiempo diario en redes sociales
- Sueño
- Estrés
- Ansiedad
- Rendimiento académico
- Actividad física
- Adicción a redes sociales
- Depresión

El objetivo principal es identificar patrones y posibles asociaciones entre los hábitos digitales y el bienestar emocional de los adolescentes.

##  DATOS

El conjunto de datos utilizado proviene de Kaggle y contiene información relacionada con hábitos digitales y salud mental en adolescentes.

### Variables principales

| Variable | Descripción |
|---|---|
| age | Edad del adolescente |
| gender | Género |
| daily_social_media_hours | Horas diarias en redes sociales |
| platform_usage | Plataforma más utilizada |
| sleep_hours | Horas de sueño |
| screen_time_before_sleep | Tiempo frente a pantalla antes de dormir |
| academic_performance | Rendimiento académico |
| physical_activity | Actividad física diaria |
| social_interaction_level | Nivel de interacción social |
| stress_level | Nivel de estrés |
| anxiety_level | Nivel de ansiedad |
| addiction_level | Nivel de adicción a redes sociales |
| depression_label | Indicador de depresión |

## HIPOTESIS ANALIZADAS

- A mayor uso de redes sociales, menor cantidad de sueño.
- A mayor tiempo frente a pantallas antes de dormir, mayor estrés.
- A mayor uso de redes sociales, menor rendimiento académico.
- A mayor uso de redes sociales, mayor nivel de adicción.
- A menor actividad física, mayor ansiedad.
- A menor actividad física, mayor depresión.

##  CONCLUSIONES

###  1. Hipótesis parcialmente confirmadas

- La actividad física parece actuar como factor protector frente a la ansiedad.
- Los adolescentes menos activos presentan mayores niveles de ansiedad.

### 2. Hipótesis no confirmadas

En este dataset no se observan relaciones fuertes entre:

- Uso intensivo de redes sociales y peor rendimiento académico.
- Uso intensivo de redes sociales y mayor estrés.
- Uso intensivo de redes sociales y mayor adicción.
- Menor actividad física y depresión.

### 3. Reflexión

El análisis muestra que muchas relaciones esperadas no aparecen de forma clara en los datos, lo que sugiere que la salud mental adolescente depende de múltiples factores y no únicamente del uso de las redes sociales.
